# sprayFlameletFoam

## Installation

  `mkdir ~/OpenFOAM/sprayFlameletFoam/`

  `git clone https://github.com/ZX114/sprayFlameletFoam.git ~/OpenFOAM/sprayFlameletFoam/`
  
  `cd ~/OpenFOAM/sprayFlameletFoam/`
  
  `./Allwmake`
## Sample
  
  `cd ~/OpenFOAM/sprayFlameletFoam/RAS`
  
  `sprayFlameletFoam`
